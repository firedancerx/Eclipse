To run one of these projects, do the following:
 - Start App Inventor. (See the instructions in the book.)
 - Visit the MyProjects screen.
 - Look for the More Options dropdown.
 - In the More Options dropdown, select Upload Source.
 - Browse to one of the .zip files in thie directory.
 - In the Upload Project dialog box, click OK.
